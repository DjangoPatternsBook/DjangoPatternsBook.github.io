Day 1

Q. Who is Django web framework named after?

A. From the FAQ, "Django is named after Django Reinhardt, a gypsy jazz guitarist from the 1930s to early 1950s." We learn from Wikipedia, that his real name was "Jean Reinhardt"; but his nickname was Django, which in Romani means "I awake."

Winner: @dommg

Day 2

Q. Which web framework was used to build Python.org?

A. On 2012 September 26th, Python Software Foundation board of directors unanimously approved the combined bids of Project Evolution / Revolution Systems and Divio to redesign Python.org. The winning bid points out that "Django framework is a no-brainer for setting up basic content editing capabilities". The site's architecture was also posted by Jacob Kaplan-Moss (RevSys).

Since Python.org source is on github, we can also check their requirements file to know the exact Django version they use, which is Django 1.7.7 currently.

Winner: @johnhubner

Day 3

Q. Django 1.8 now supports which 3rd party template engine?

A. On Oct 1, 2014 Django core team member Aymeric Augustin launched an IndiGogo campaign to provide first-class support for third-party template engines in Django and native Jinja2 integration. The campaign was fully funded in just two days.

By Jan 25, 2015 Aymeric completed all his project goals just in time for Django 1.8 alpha. Django 1.8 was released with built-in support for the Django template language and for Jinja2.

Winner: @ananthrk

Day 4

Q. Which was the first Django release to get support for Python 3?

A. Django 1.5 introduced experimental support for Python 3 - specifically, Python 3.2 and above. By Django 1.6, Python 3 was officially supported and recommended for production.

Winner: @k4rtik
